AI-Powered-Financetracker

Project Date April 2025
Role FullStack Developer, Data Analyst
Visit https://github.com/ShamKumarM/AI-Powered-FinanceTracker

The Expense Tracker project is a comprehensive web application developed using Django for the backend and React for the frontend. It enables users to manage their personal finances efficiently by logging expenses, categorizing transactions, and visualizing financial data. The application incorporates user authentication for secure access, real-time currency conversion using a currency API, and database integration using SQLite for persistent data storage. Additionally, the project utilizes a preloaded dataset and natural language processing (NLP) techniques to automate the classification of expense descriptions, enhancing user convenience and minimizing manual data entry.

A standout feature of this project is its integration of machine learning to analyze usersâ€™ financial behavior and provide insights into their spending patterns. By identifying trends and offering predictive suggestions, the application helps users make better budgeting decisions. The intuitive user interface built with React ensures smooth navigation and an enhanced user experience. This system not only simplifies the process of tracking expenses but also empowers individuals with smart financial tools to improve personal money management and long-term financial planning
